# 블로그 초안 작성 가이드 (기준: opus-vs-fable-seat-count.mdx, 2026-07-06 확정)

첫 초안(Opus vs Fable 회고)에서 사용자 피드백으로 확정된 규칙. 모든 신규 초안은 이 가이드를 따른다.

## 1. 구조 템플릿

- **frontmatter**: `title`(주제 — 부제 형태, 구체적 수치·대비 포함 가능), `description`(~합니다 톤 한 줄, 핵심 수치나 반전 포함), `date`(실제 작업일로 소급), `tags`(배열), `cover: null`
- **도입부**: 인사말·자기소개 없이 시점+상황으로 바로 시작. 도입 마지막 문장에서 글의 핵심(반전·주제)을 미리 제시하되 "미리 말해두면" 같은 극적 구문은 금지
- **"어떤 문제였나"류 섹션**: 문제 정의 + 처음의 (안일한) 가정 → 실제 난점 순서
- **본문**: 시간순 서사. 시각·날짜·버전 번호 등 타임스탬프 근거를 구체적으로 (예: "16시 9분에 v5가 나왔고 3분 뒤에 v6이")
- **비교·정리가 필요하면 GFM 표** 사용 (remark-gfm)
- **"배운 것" 섹션**: "첫째, 둘째, 셋째" 문단형 교훈. 각 교훈은 사건 근거 → 일반화 순서
- **마무리**: `---` 구분선 후 짧은 코다. **"이 접근이 어디에 일반화되는가" 확장 단락 필수** (포트폴리오 서사)

## 2. 문체 (plain-writing-style — 위반 시 반려)

- 경어체: ~합니다/~입니다/~됩니다
- 전문용어는 쓰되 풀어쓰기 동반: "영상 전체를 한 장의 큰 이미지로 이어 붙였습니다(스티칭)"
- 압축 명사구 대신 문장: "색 임계값 탈락" ❌ → "기준치에 미달해 통째로 걸러지고 있던 것" ✅
- 한자어 줄이기: 이월→미룸, 격리→따로 빼둠, 국소 수정→부분 수정, 실측→실제로 재보기
- 인용구가 함축적이면 바로 다음 줄에 한 줄 풀이

### AI 냄새 나는 화법 금지 목록
- 본문 중간 볼드 강조 남발 금지
- 엠대시(—) 남발 금지 (제목의 "주제 — 부제" 1회는 허용)
- 극적 반전 구문 금지: "결과만 말하면", "미리 말해두면", "진짜 이야기는 ~에 있습니다"
- 겉멋 은유 금지: "패배 선언", "구조의 그림자" 류
- 3연속 병렬 인용 나열 금지
- "~였으니까요/~던 겁니다" 어미 반복 금지
- 소제목은 직설·질문형: "어떤 문제였나", "색으로 세면 되겠는데?" ✅ / 명사형 멋부림("남는 것"), 대구형("패치를 쌓을 것인가, 문제를 지울 것인가") ❌
- 코드 삽입은 꼭 필요할 때만 (회고 글엔 대부분 불필요)
- "공정한 비교인가?" 같은 별도 방어 섹션 금지 — 필요하면 서사 속 한두 문장으로
- 사용자(작성자) 탓으로 읽히는 프레임 금지

## 3. 사실 검증

- 모든 수치·날짜·시각은 원자료(git log, 세션 로그, 파일 타임스탬프)에서 확인한 것만 단정 서술
- 원자료에서 확인 못 한 내용은 본문에 쓰지 말거나, 꼭 필요하면 `{/* ⚠️ 미확인: ... */}` MDX 주석으로 표시
- AI가 그럴듯하게 지어낸 서사 금지 — 원자료에 없는 대화·감정·시각은 만들지 않는다

## 4. 익명화 (전 글 공통, 위반 시 발행 불가)

- 회사·서비스명 익명화: 하이버, 마미, 크루 어드민, 브랜디 등 → "이커머스 서비스", "어드민" 등으로 일반화
- 동료 실명·닉네임 금지 (예: martin)
- 익명 운영 블로그와의 연결 금지: 블로그명(소확행, afterworks, mix-note, daily-kbo, justaskall, bbugook 등), 필명, URL, 네이버 발행 ID 절대 노출 금지 → "운영 중인 블로그" 정도로만
- CloudWatch RUM 소재 금지 (동료 작성물)

## 5. 이미지

- 경로: `/images/<slug>/파일명.png` (영문 파일명)
- 이미지가 아직 없으면 본문 해당 위치에 `{/* TODO 이미지: <무엇을 캡처/제작할지 구체 설명> */}` 주석으로 표시
- 캡처 원본에 회사·계정 정보가 있으면 가림 처리 전제로 설명 작성
- alt 텍스트는 서술형 한 문장 (첫 초안 예: "첫 버전이 세어낸 419구역. 초록 테두리가 검출된 좌석입니다")

## 6. 분량

- 소재 밀도에 맞게. 기준 초안은 한글 약 7,000자(공백 포함). 서사가 얇으면 억지로 늘리지 말 것
