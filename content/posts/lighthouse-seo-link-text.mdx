## 링크에 대한 명확한 텍스트가 없습니다

> 명확한 링크 텍스트는 검색엔진이 컨텐츠를 이해하는 데 도움이 됩니다.

링크 텍스트는 하이퍼링크 내에서 클릭 가능한 단어를 의미하며, 사용자를 새로운 페이지로 안내합니다.

### 문제가 되는 경우

Lighthouse는 다음과 같은 모호한 링크 문구를 식별합니다:

- "click here"
- "click this"
- "go"
- "here"
- "start"
- "right here"
- "more"
- "learn more"

### 해결 방법

일반적인 표현 대신 목적지 콘텐츠를 전달하는 설명적 텍스트를 사용하세요.

**나쁜 예:** "농구 동영상을 보려면 **여기를 클릭**하세요."

**좋은 예:** "**농구 동영상**을 확인해보세요."

### 모범 사례

- 관련 주제에 초점을 맞출 것
- URL 자체를 링크 텍스트로 사용하지 말 것
- 설명을 간결하게 유지할 것
- 내부 링크 품질을 개선할 것

### 참고

현재 Lighthouse의 필터는 영어에만 작동하며 한국어에는 적용되지 않습니다. 다만 "더보기" 같은 표현을 사용하는 사이트는 어떤 콘텐츠가 확장되는지 명시하여 웹 접근성을 준수하는 것이 좋습니다.
