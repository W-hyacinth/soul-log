약 1년간 SEO 관련 업무를 진행하며 경험한 사례를 공유합니다. Google Lighthouse에서 SEO 지표가 100이지만, 실제 검색 결과는 설정한 meta description과 다르게 노출되는 현상이 발생했습니다.

## Google의 description 노출 패턴 가설

1. 검색 키워드가 description에 모두 포함되면 해당 description 노출
2. 키워드가 포함되지 않거나 더 적절한 컨텐츠가 있으면 컨텐츠 내용 노출
3. 부분적으로 일치하면 description과 컨텐츠 혼합 노출
4. 어느 쪽에도 포함되지 않으면 설정된 description 노출

## 3. 결과

description에 서비스명을 포함시킨 후 검색하니 설정된 description이 정상 노출되었습니다.
