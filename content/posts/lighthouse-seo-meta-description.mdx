## 문서에 `<meta name="description">`이 없습니다

> Meta description elements allow search engines to include summarized page content in results.

`<meta name="description">` 요소는 검색 엔진이 검색 결과에 페이지 요약을 표시할 수 있게 해줍니다. 품질 높고 고유한 description은 가시성을 높이고 유기적 트래픽을 증가시킬 수 있습니다.

### 해당 항목에서 통과하지 못한 이유

Lighthouse는 다음의 경우 실패로 식별합니다:

- 페이지에 `<meta name="description">` 요소가 아예 없는 경우
- `content` 속성이 비어있는 경우

Lighthouse는 description의 품질은 평가하지 않고, 존재 여부만 확인합니다.

### meta description 추가 방법

페이지의 `<head>`에 다음 요소를 삽입합니다:

```html
<meta name="description" content="여기에 설명을 작성합니다">
```

### 모범 사례

- 페이지마다 고유한 description 작성
- 명확하고 간결하게 작성, "Home" 같은 모호한 표현 지양
- 키워드 채우기 금지 — 사용자에게 해롭고 검색엔진의 스팸 필터에 걸릴 수 있음
- 적절한 경우 관련 세부정보 포함 (작성자, 카테고리, 가격 등)

**나쁜 예:** "도넛 레시피" (너무 모호함)

**좋은 예:** "메리의 간편한 메이플 베이컨 도넛 레시피는 숨겨진 복잡함과 끈적한 달콤함을 자랑합니다."
